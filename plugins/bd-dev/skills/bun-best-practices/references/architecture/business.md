## 业务领域

使用 Mermaid mindmap 划分业务领域：

```mermaid
mindmap
  root((业务领域))
    子领域A
      概念1
      概念2
    子领域B
      概念3
      概念4
```

## 核心流程

使用 Mermaid flowchart 绘制核心业务流程：

```mermaid
flowchart LR
    A[开始] --> B{条件判断}
    B -->|是| C[流程1]
    B -->|否| D[流程2]
    C --> E[结束]
    D --> E
```
