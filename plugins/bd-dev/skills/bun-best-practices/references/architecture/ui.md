# UI 布局架构模板

前端项目使用此模板描述 UI 布局架构。

## 页面结构

使用 ASCII 绘制页面结构：

```
┌─────────────────────────────────────┐
│              Header                  │  ← 顶部导航
├──────────────┬──────────────────────┤
│              │                      │
│   Sidebar   │      Content         │  ← 主体区域
│   (侧边栏)   │      (主内容)        │
│              │                      │
│              │                      │
├──────────────┴──────────────────────┤
│              Footer                  │  ← 底部信息
└─────────────────────────────────────┘
```

## 响应式断点

| 断点    | 宽度       | 布局变化         |
| ------- | ---------- | ---------------- |
| mobile  | < 768px    | 单列，隐藏侧边栏 |
| tablet  | 768-1024px | 双列，侧边栏折叠 |
| desktop | > 1024px   | 多列，显示侧边栏 |

## 组件层级

使用 ASCII 绘制组件层级：

```
App
└── Layout
    ├── Header
    │   ├── Logo
    │   ├── Nav
    │   └── Actions
    ├── Main
    │   ├── Sidebar
    │   │   ├── Menu
    │   │   └── Links
    │   └── Content
    │       ├── PageHeader
    │       └── PageBody
    └── Footer
```

## 路由结构

| 路径       | 页面   | 布局            |
| ---------- | ------ | --------------- |
| /          | 首页   | DefaultLayout   |
| /about     | 关于   | DefaultLayout   |
| /dashboard | 仪表盘 | DashboardLayout |

## 布局模式

### 经典布局

```
┌─────────────────────────────────────┐
│              Header                  │
├──────────────┬──────────────────────┤
│   Sidebar   │      Content         │
│              │                      │
└──────────────┴──────────────────────┘
```

### 仪表盘布局

```
┌─────────────────────────────────────┐
│              Header                  │
├──────────────┬──────────────────────┤
│              │    ┌───────────┐    │
│   Sidebar   │    │  Widget1  │    │
│              │    ├───────────┤    │
│              │    │  Widget2  │    │
│              │    ├───────────┤    │
│              │    │  Widget3  │    │
│              │    └───────────┘    │
└──────────────┴──────────────────────┘
```

### 全屏布局

```
┌─────────────────────────────────────┐
│              Header                  │
├─────────────────────────────────────┤
│                                     │
│            Content                  │
│           (全屏宽度)                 │
│                                     │
└─────────────────────────────────────┘
```
