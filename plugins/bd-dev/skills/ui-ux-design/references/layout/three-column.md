# 三栏布局

左导航 + 主内容 + 右面板的布局结构。

---

## 定义

三栏布局在双栏布局的基础上增加右侧面板，常用于需要同时展示列表和详情，或需要辅助信息的场景。

---

## 结构

```
┌─────────────────────────────────────────────────────┐
│                      Header                          │
├────────┬────────────────────────────┬───────────────┤
│        │                            │               │
│  Nav   │        Main Content        │    Panel      │
│ 200px  │         (flex: 1)         │   280px      │
│        │                            │               │
│ - Item │   ┌──────────────────┐    │  Details      │
│ - Item │   │    Content        │    │  Actions     │
│ - Item │   │                   │    │  Info        │
│        │   └──────────────────┘    │               │
│        │                            │               │
├────────┴────────────────────────────┴───────────────┤
│                      Footer                          │
└─────────────────────────────────────────────────────┘
```

---

## 适用场景

- 邮件客户端（邮件列表 + 邮件内容）
- 聊天界面（联系人列表 + 聊天窗口 + 详情）
- 文档编辑器（大纲 + 文档 + 目录）
- 企业后台（导航 + 列表 + 详情）

---

## 变体

### 左右固定，中间自适应

```
┌──────┬──────────────────────────┬──────┐
│ Nav  │       Content            │Panel │
│ 200px│         flex: 1          │280px │
└──────┴──────────────────────────┴──────┘
```

### 左固定，中间和右自适应

```
┌──────┬──────────────────────────┬──────┐
│ Nav  │       Content            │Panel │
│ 200px│         flex: 1          │flex:1│
└──────┴──────────────────────────┴──────┘
```

---

## 响应式行为

| 屏幕宽度  | 行为                   |
| --------- | ---------------------- |
| < 768px   | 只显示主内容，其他隐藏 |
| 768-992px | 显示左导航 + 内容      |
| ≥ 992px   | 三栏全部显示           |

---

## 禁忌

- ❌ 移动端体验差，需要慎重设计
- ❌ 信息过载，三个区域都很拥挤
- ❌ 不适合简单页面
