# 双栏布局

侧边栏 + 主内容区的经典布局结构。

---

## 定义

双栏布局将页面分为左侧固定宽度的侧边栏和右侧主内容区域，侧边栏通常用于导航或辅助信息。

---

## 结构

```
┌─────────────────────────────────────────────┐
│                   Header                    │
├────────────┬────────────────────────────────┤
│            │                                │
│  Sidebar   │         Main Content           │
│  (200-280px)│                                │
│            │                                │
│  - Nav 1   │   ┌─────────────────────┐      │
│  - Nav 2   │   │     Content         │      │
│  - Nav 3   │   │                     │      │
│            │   └─────────────────────┘      │
│            │                                │
├────────────┴────────────────────────────────┤
│                   Footer                    │
└─────────────────────────────────────────────┘
```

---

## 适用场景

- 管理后台 (Dashboard)
- 个人中心
- 设置页面
- 文档列表页

---

## 侧边栏类型

### 固定侧边栏

- 宽度固定（200-280px）
- 位置固定，不随滚动移动
- 适合主要导航功能

### 可折叠侧边栏

- 可以展开/收起
- 收起时只显示图标
- 适合功能繁多的后台系统

---

## 响应式行为

| 屏幕宽度 | 行为                 |
| -------- | -------------------- |
| < 768px  | 侧边栏隐藏，改为抽屉 |
| ≥ 768px  | 侧边栏显示           |
| ≥ 992px  | 侧边栏宽度 240px     |

---

## 禁忌

- ❌ 侧边栏内容过多
- ❌ 主内容需要宽屏展示的场景
- ❌ 移动端优先的简单页面

---

## CSS 实现示例

```css
.layout {
  display: flex;
  min-height: 100vh;
}

.sidebar {
  width: 240px;
  flex-shrink: 0;
}

.main {
  flex: 1;
  min-width: 0;
}

@media (max-width: 768px) {
  .sidebar {
    display: none;
  }
}
```
