# 瀑布流布局

不规则高度的卡片垂直堆叠的布局方式。

---

## 定义

瀑布流布局（Waterfall / Masonry Layout）让内容卡片像水流一样自然填充空间，每列高度不一致但间距均匀。适合展示高度不一的图片或内容。

---

---

## 适用场景

- 图片画廊
- Pinterest 风格瀑布流
- 社交媒体动态流
- 电商商品列表
- 瀑布流新闻

---

## 实现方式

### CSS Grid (多列)

```css
.masonry {
  column-count: 4;
  column-gap: 16px;
}

.masonry-item {
  break-inside: avoid;
  margin-bottom: 16px;
}
```

### Flexbox + JavaScript

通过 JavaScript 计算每列高度，动态分配项目到最短列。

### CSS Grid + JavaScript

使用 Grid 布局配合 JavaScript 实现更精确的控制。

---

## 响应式行为

| 屏幕宽度 | 列数 |
| -------- | ---- |
| < 576px  | 2 列 |
| ≥ 576px  | 3 列 |
| ≥ 768px  | 4 列 |
| ≥ 1200px | 5 列 |

---

## 性能考虑

- 图片懒加载
- 虚拟滚动（大量数据时）
- 使用 CSS `content-visibility: auto`

---

## 禁忌

- ❌ 不适合需要严格对齐的场景
- ❌ 不适合文字为主的内容
- ❌ 不适合需要精确控制的表格数据
