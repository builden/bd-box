# Pagination - 分页

分页器用于长列表的分页导航。

---

## 核心设计考量

### 分页模式

| 模式     | 适用场景      |
| -------- | ------------- |
| 页码     | 数据量 < 1000 |
| 游标     | 数据量 > 1000 |
| 加载更多 | 移动端        |

### 页码结构

```
« ‹ 1 2 3 ... 10 › »
```

---

## 交互行为

- 首页/末页按钮
- 上一页/下一页
- 直接跳转页码
- 每页条数选择

---

## 状态定义

- **Default**：正常状态
- **Hover**：悬停高亮
- **Active**：当前页
- **Disabled**：首尾页不可点

---

## 动效规范

- 切换页码：300ms 列表刷新
- 条数切换：200ms

---

## 可访问性

- 需支持键盘左右箭头切换页码
- 需告知总页数
