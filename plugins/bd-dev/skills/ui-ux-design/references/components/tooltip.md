# Tooltip - 提示

用于解释性文本的轻量级浮层，鼠标悬停或获得焦点时显示。

---

## 定义

Tooltip（提示）是一个小型的浮层，当用户鼠标悬停在元素上或元素获得焦点时显示，用于提供解释性信息或补充说明。

---

## 典型场景

- 图标按钮的功能解释
- 缩写词的完整形式
- 省略文字的完整展示
- 复杂操作的简短说明

---

## 核心设计考量

### 1. 内容简洁

- 纯文本为主
- 避免复杂格式
- 建议不超过一行

### 2. 位置自动调整

- 优先显示在合适的位置
- 避免遮挡目标元素
- 避免超出视口

### 3. 显示时机

- 鼠标悬停后延迟显示（通常 100-200ms）
- 鼠标移开后延迟隐藏
- 避免闪烁

---

## 位置定义

| 位置        | 说明           |
| ----------- | -------------- |
| top（默认） | 显示在元素上方 |
| bottom      | 显示在元素下方 |
| left        | 显示在元素左侧 |
| right       | 显示在元素右侧 |

---

## 交互行为

### 显示

1. 鼠标悬停 / 获得焦点
2. 延迟 100-200ms
3. Tooltip 渐现 + 微位移

### 隐藏

1. 鼠标移开 / 失去焦点
2. 延迟 100ms
3. Tooltip 渐隐

---

## 动效规范

### 显示

- 持续时间：100ms
- 曲线：ease-out
- 效果：opacity(0 → 1) + translateY(4px → 0)

### 隐藏

- 持续时间：50ms
- 曲线：ease-in
- 效果：opacity(1 → 0)

---

## 样式规范

- 背景色：rgba(0, 0, 0, 0.75)
- 文字色：#fff
- 字号：12px
- 内边距：8px 12px
- 圆角：4px
- 最大宽度：250px

---

## 可访问性

- 键盘可触发（Tab 键）
- 屏幕阅读器可见（aria-describedby）
- 不包含关键信息（仅辅助说明）

---

## 禁忌

- ❌ 重要信息不要只放在 Tooltip 中
- ❌ 避免过长内容
- ❌ 交互操作不要放在 Tooltip 中
- ❌ 避免在触摸设备上依赖 Tooltip
