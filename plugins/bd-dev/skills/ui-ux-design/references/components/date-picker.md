# DatePicker - 日期选择器

日期选择器用于选择日期或日期范围。

---

## 核心设计考量

### 类型

| 类型    | 用途     |
| ------- | -------- |
| Date    | 单日选择 |
| Week    | 周选择   |
| Month   | 月选择   |
| Quarter | 季度选择 |
| Year    | 年选择   |
| Range   | 日期范围 |

### 面板结构

```
◀ 2024年3月 ▶
日 一 二 三 四 五 六
───────1───2───3───
4───5───6───7───8───
...
```

---

## 状态定义

- **Default**：正常显示
- **Hover**：日期高亮
- **Selected**：选中日期填充
- **Range**：范围日期区间填充
- **Disabled**：不可选日期灰色
- **Today**：今日标记

---

## 交互行为

- 点击输入框展开面板
- 选择日期后自动关闭
- 支持快捷操作（今天、昨天）
- 范围选择需支持左右面板联动

---

## 动效规范

- 面板展开：200ms 缩放 + 淡入
- 年月切换：300ms 滑动

---

## 可访问性

- 需支持键盘导航日期
- 需告知选中日期的语义
- 需提供清空按钮

---

## 禁忌

- 不要用于模糊时间（用输入框）
- 不要在移动端使用复杂面板
