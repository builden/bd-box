# Incremental Static Rendering

## 一句话定义

增量静态生成，部分页面静态，部分动态。

## 为什么好

- **混合策略**：结合 SSG 和 SSR
- **按需生成**：只生成需要的页面
- **更新灵活**：部分页面可以实时更新

## 适用场景

- 大型网站
- 电商网站
- 内容管理系统

## 框架支持

- Next.js ISR
- Nuxt
