# RSC (React Server Components)

## 一句话定义

React 组件只在服务端渲染，不打包到客户端。

## 为什么好

- **零 bundle**：服务端组件不增加 bundle
- **直接访问**：服务端直接访问数据库/文件系统
- **自动划分**：服务端/客户端自动分离

## 适用场景

- 数据获取
- 大组件
- 第三方库

## 框架支持

- Next.js 13+
- Remix
