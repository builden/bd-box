# 数据分层架构模板

有状态项目使用此模板描述数据分层和数据流。

## 数据流图

使用 Mermaid flowchart 绘制数据流向：

```mermaid
flowchart LR
    A[原始数据] --> B[数据校验]
    B --> C{校验结果}
    C -->|通过| D[派生计算]
    C -->|失败| E[错误处理]
    D --> F[缓存]
    D --> G[持久化]
```
