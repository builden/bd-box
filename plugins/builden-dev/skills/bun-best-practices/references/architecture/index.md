# 项目规范文档

项目启动时创建的架构、数据流、UI 布局等规范文档。

## 目录

| 文档                       | 说明             |
| -------------------------- | ---------------- |
| [business.md](business.md) | 业务架构模板     |
| [system.md](system.md)     | 系统架构模板     |
| [data.md](data.md)         | 数据分层架构模板 |
| [ui.md](ui.md)             | UI 布局架构模板  |

## 文档位置

```
docs/
└── specs/              # 规范文档存放目录
    ├── architecture/   # 架构文档
    ├── data/          # 数据架构
    └── ui/            # UI 架构
```

## 何时创建

- **项目初始化时**：创建基础架构文档
- **功能开发中**：补充模块子架构、更新数据流
- **重构/大改**：更新相关架构文档
- **Code Review**：检查文档与实现是否一致

## 文档特性

- 使用 Mermaid 绘制架构图
- AI 可读（纯文本 + 代码块）
- 人类可维护（简洁模板，避免过度设计）
