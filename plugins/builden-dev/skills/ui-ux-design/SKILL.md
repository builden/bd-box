# UI/UX/动效设计 Skill

本 skill 帮助开发者在实现 UI 组件时参考业界优秀设计体系，获取具体的组件设计模式、布局结构和动效规范。

## 触发场景

- "这个组件怎么设计"
- "用什么布局好"
- "动画曲线怎么选"
- "交互反馈怎么做"
- 提及具体组件名（Modal、Drawer、Tooltip 等）
- 提及设计价值观（一致性、反馈等）

## 文档结构

每个设计模式包含：

- 一句话定义
- 为什么好
- 适用场景
- 禁忌（什么时候不该用）
- 交互规范
- 动效规范
- 可访问性要求
