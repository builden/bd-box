# Table - 表格

表格用于展示结构化数据，支持排序、分页、筛选等交互。

---

## 核心设计考量

### 基本结构

```
┌─────────────────────────────────────────┐
│  表头（可选：排序、筛选）                 │
├─────────────────────────────────────────┤
│  行数据                                  │
│  行数据                                  │
│  行数据                                  │
├─────────────────────────────────────────┤
│  分页器                                   │
└─────────────────────────────────────────┘
```

### 核心功能

| 功能   | 说明          |
| ------ | ------------- |
| 排序   | 单列/多列排序 |
| 筛选   | 筛选器下拉    |
| 分页   | 页码/加载更多 |
| 行选择 | Checkbox 选择 |
| 列固定 | 左右固定      |

---

## 状态定义

- **Default**：正常显示
- **Hover**：行高亮
- **Selected**：选中行背景
- **Loading**：骨架屏
- **Empty**：空状态占位

---

## 交互行为

### 行操作

- Hover 显示操作列
- 点击行跳转详情
- 右键菜单（可选）

### 排序交互

```css
.sort-icon {
  opacity: 0;
  transition: opacity 0.2s;
}
th:hover .sort-icon {
  opacity: 0.5;
}
th.sorted .sort-icon {
  opacity: 1;
}
```

---

## 动效规范

- 行 Hover：150ms 背景过渡
- 列切换：300ms 滑动
- Loading：骨架屏脉冲动画

---

## 可访问性

- 表头需 `scope="col"`
- 需支持键盘导航
- 排序需告知状态变化

---

## 禁忌

- 不要在表格中放置过多列（横向滚动影响体验）
- 不要省略空状态
- 不要用表格做布局
