# 单栏布局

内容垂直居中，最大宽度限制的布局方式。

---

## 定义

单栏布局将主要内容放置在页面中心，内容区域有最大宽度限制，两侧留白。适合内容驱动的页面。

---

## 结构

```
┌─────────────────────────────────────┐
│              Header                │
├─────────────────────────────────────┤
│                                     │
│         ┌───────────────┐           │
│         │   Main        │           │
│         │   Content     │           │
│         │               │           │
│         └───────────────┘           │
│                                     │
├─────────────────────────────────────┤
│              Footer                │
└─────────────────────────────────────┘
```

---

## 适用场景

- 博客文章页
- 文档页面
- 新闻详情页
- 个人网站首页
- 注册/登录页

---

## 特征

- 内容居中显示
- 最大宽度限制（通常 600-900px）
- 两侧留白随屏幕宽度自适应
- 适合线性阅读

---

## 响应式行为

| 屏幕宽度 | 行为                   |
| -------- | ---------------------- |
| < 768px  | 全宽，移除两侧 padding |
| ≥ 768px  | 居中，内容宽度 720px   |
| ≥ 1200px | 居中，内容宽度 900px   |

---

## 禁忌

- ❌ 不适合信息密度高的页面
- ❌ 不适合需要并排对比的场景
- ❌ 不适合复杂的 Dashboard

---

## CSS 实现示例

```css
.container {
  max-width: 900px;
  margin: 0 auto;
  padding: 0 16px;
}

@media (max-width: 768px) {
  .container {
    padding: 0;
  }
}
```
