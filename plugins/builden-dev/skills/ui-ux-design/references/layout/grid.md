# 栅格布局

基于列的网格系统，内容按列排列的布局方式。

---

## 定义

栅格布局使用列（Column）和行（Row）的概念，将页面划分为等宽或不等宽的列，内容在列中排列。是响应式设计的基础。

---

## 结构

```
┌─────────────────────────────────────────────────────┐

└─────────┴─────────┴─────────┴─────────┴───────────┘
```

---

## 适用场景

- 产品列表页
- 卡片式内容展示
- 数据仪表盘
- 图片画廊
- 表单布局

---

## 核心概念

### 基础栅格

- **24 栏系统**：将宽度分为 24 等份
- **Gutter**：列之间的间距（通常 16px 或 24px）
- **Margin**：栅格与容器边缘的间距

### 响应式列数

| 断点 | 列数  |
| ---- | ----- |
| xs   | 2 列  |
| sm   | 4 列  |
| md   | 8 列  |
| lg   | 12 列 |
| xl   | 24 列 |

---

## 响应式行为

```css
.row {
  display: flex;
  flex-wrap: wrap;
  margin: 0 -8px;
}

.col-24 {
  flex: 0 0 100%;
  padding: 0 8px;
}
.col-12 {
  flex: 0 0 50%;
  padding: 0 8px;
}
.col-8 {
  flex: 0 0 33.33%;
  padding: 0 8px;
}
.col-6 {
  flex: 0 0 25%;
  padding: 0 8px;
}

@media (max-width: 576px) {
  .col-12,
  .col-8,
  .col-6 {
    flex: 0 0 100%;
  }
}
```

---

## 间距规范

| 名称 | 间距 |
| ---- | ---- |
| xs   | 4px  |
| sm   | 8px  |
| md   | 16px |
| lg   | 24px |
| xl   | 32px |

---

## 禁忌

- ❌ 单一页面大量使用会显得零散
- ❌ 列数过多（超过 24 栏）
- ❌ 跨栏内容在不同断点下表现不一致
