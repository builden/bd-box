# 布局设计模式索引

本文档整理常见的页面布局结构，帮助开发者选择合适的布局方案。

---

## 布局模式总览

| 模式                         | 描述                       | 典型场景            | 复杂度 |
| ---------------------------- | -------------------------- | ------------------- | ------ |
| [单栏布局](single-column.md) | 内容垂直居中，最大宽度限制 | 博客、文档、文章页  | 简单   |
| [双栏布局](two-column.md)    | 侧边栏 + 主内容区          | Dashboard、后台管理 | 简单   |
| [三栏布局](three-column.md)  | 左导航 + 内容 + 右面板     | 邮件、聊天、文档    | 中等   |
| [栅格布局](grid.md)          | 列式网格系统               | 卡片列表、产品矩阵  | 中等   |
| [瀑布流布局](waterfall.md)   | 不规则高度垂直堆叠         | 图片流、社交媒体    | 中等   |
| [圣杯布局](holy-grail.md)    | 固定 header/footer + 三栏  | 复杂管理后台        | 复杂   |
| [F/Z 形布局](f-z-layout.md)  | 阅读路径视觉引导           | Landing Page、首页  | 简单   |

---

## 选择指南

### 按场景选择

- **内容阅读为主** → 单栏布局
- **功能操作为主** → 双栏/三栏布局
- **信息密度高** → 栅格布局
- **视觉导向** → 瀑布流/F/Z 形布局

### 按复杂度选择

- **简单页面**：单栏、F/Z 形
- **中等复杂度**：双栏、三栏、栅格、瀑布流
- **复杂后台**：圣杯布局

## 布局要素

### 留白（Whitespace）

- 内容区域与边缘保持适当间距
- 相关元素之间使用一致的间距
- 使用 8px 基础网格系统

### 对齐（Alignment）

- 元素与网格对齐
- 文本基线对齐
- 视觉元素边缘对齐

### 层级（Hierarchy）

- 通过间距和大小表达层级
- 标题与正文的比例关系
- 颜色深浅表达重要性

## 典型应用场景

### 单栏布局

**典型网站**：Medium、知乎文章、博客、个人网站、新闻详情页

**采用原因**：

- **阅读体验**：内容居中，两侧留白，视线移动距离最短
- **内容为王**：核心价值是文章/内容，不需要复杂导航
- **简洁美观**：最大宽度限制让内容保持优雅的阅读宽度

### 双栏布局

**典型网站**：企业官网、WordPress、简单管理后台

**采用原因**：

- **功能导向**：需要同时展示导航和内容
- **操作频繁**：侧边栏承载导航，内容区承载操作
- **效率优先**：用户需要快速在不同功能间切换

### 三栏布局

**典型网站**：Gmail、Slack、Notion、文档编辑器

**采用原因**：

- **多任务并行**：左侧导航 + 中间列表 + 右侧详情
- **信息密度高**：需要同时查看多个信息块
- **效率导向**：专业工具，追求操作效率

### 栅格布局

**典型网站**：电商商品列表、Dribbble、Behance

**采用原因**：

- **内容模块化**：内容本身是独立的卡片/块
- **灵活排列**：列数可响应式调整
- **信息丰富**：展示大量相似内容，高效利用空间

### 瀑布流布局

**典型网站**：Pinterest、Instagram、淘宝/天猫首页

**采用原因**：

- **视觉驱动**：图片为主，高度不规则
- **探索体验**：用户"刷"内容，像瀑布一样源源不断
- **沉浸感**：无边界的内容流，让用户持续浏览

### 圣杯布局

**典型网站**：复杂后台管理系统（SaaS）、ERP、CRM

**采用原因**：

- **功能复杂**：多导航、多操作、多面板
- **层级清晰**：Header（全局）+ 侧边栏（功能）+ 主内容 + 右侧（辅助）
- **专业工具**：面向管理员，需要高信息密度

## 选择决策

**一句话原则**：先问自己——用户来这个网站/应用是做什么的？

| 核心需求 | 推荐布局    |
| -------- | ----------- |
| 看内容   | 单栏        |
| 做操作   | 双栏/三栏   |
| 探索发现 | 栅格/瀑布流 |
| 管理控制 | 圣杯        |
