# 数据结构和算法 Skill

数据结构和算法知识指南，帮助开发者选择合适的数据结构和算法。

## 使用场景

- "用什么数据结构好"
- "这个场景用什么算法"
- "想优化性能"
- 提及具体数据结构名（数组、链表、LRU、B+树等）
- 提及具体算法名（快排、动态规划、A\*等）

## 文档分类

### 按领域

- 缓存应用：LRU、LFU、布隆过滤器
- 关系型数据库：B+树
- NoSQL 数据库：LSM树、跳表、B树
- 分布式数据库：一致性哈希、Gossip、Raft
- 计算应用：数组、链表、栈、队列、哈希表、树、堆、图
- AI 应用：向量检索、KD树、HNSW
- 游戏应用：四叉树、八叉树、A\*、对象池
- 系统设计：令牌桶、漏桶
