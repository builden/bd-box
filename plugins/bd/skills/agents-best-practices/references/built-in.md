# 内置 Subagents

Claude Code 内置以下 Subagents，每个都继承主对话的权限并有额外工具限制。

## Explore

快速、只读的代码库搜索和分析代理。

- **模型**：Haiku（快速、低延迟）
- **工具**：只读工具（拒绝 Write 和 Edit）
- **用途**：文件发现、代码搜索、代码库探索

调用时指定详细程度：**quick**（目标查找）、**medium**（平衡探索）、**very thorough**（全面分析）

## Plan

计划模式下进行代码库研究的代理。

- **模型**：继承主对话
- **工具**：只读工具
- **用途**：代码研究

## General-purpose

用于复杂多步骤任务的通用代理。

- **模型**：继承主对话
- **工具**：全部工具
- **用途**：复杂研究、多步骤操作、代码修改

## 其他内置代理

| Agent             | 模型   | 使用场景                  |
| ----------------- | ------ | ------------------------- |
| Bash              | 继承   | 在独立上下文运行终端命令  |
| statusline-setup  | Sonnet | 配置状态行                |
| Claude Code Guide | Haiku  | 回答 Claude Code 功能问题 |

## 自动委托

Claude 根据任务描述和 Subagent 配置中的 `description` 字段自动决定何时委托。鼓励主动委托在描述字段中使用"主动使用"等短语。
