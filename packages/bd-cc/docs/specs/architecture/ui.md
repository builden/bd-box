# UI 布局架构

CloudCLI (Claude Code UI) 项目的 UI 布局架构文档。

## 页面结构

### 桌面端布局

```
┌──────────────────────────────────────────────────────────────────────┐
│  Header (顶部栏)                                                    │
│  ┌─────────────────────────────────────────────────────────────────┐ │
│  │ Logo │ Provider 选择 │ 项目选择    │ Settings │ User Avatar    │ │
│  └─────────────────────────────────────────────────────────────────┘ │
├────────────┬─────────────────────────────────────────┬───────────────┤
│            │                                         │               │
│  Sidebar   │           Main Content                 │  Right Panel  │
│  (侧边栏)  │           (主内容区)                  面板)   │
 │  (右侧│            │                                         │               │
│  ┌──────┐  │  ┌─────────────────────────────────┐  │  ┌─────────┐  │
```

## 响应式断点

| 断点    | 宽度       | 布局变化                         |
| ------- | ---------- | -------------------------------- |
| mobile  | < 768px    | 单列布局，底部导航，隐藏右侧面板 |
| tablet  | 768-1024px | 双列布局，侧边栏可折叠           |
| desktop | > 1024px   | 多列布局，显示所有面板           |

## 路由结构

| 路径                  | 页面       | 布局       |
| --------------------- | ---------- | ---------- |
| `/`                   | 主界面     | MainLayout |
| `/login`              | 登录页     | AuthLayout |
| `/setup`              | 初始设置页 | AuthLayout |
| `/settings`           | 设置页     | Modal      |
| `/project/:projectId` | 项目详情   | MainLayout |

## 布局模式

### 主布局 (MainLayout)

```
┌─────────────────────────────────────────────────────────────────┐
│ Header (固定高度 56px)                                           │
├────────────┬────────────────────────────────────┬───────────────┤
│            │                                    │               │
│  Sidebar   │        Main Content               │ Right Panel   │

│  - Chat    │  ┌────────────────────────────┐  │  - Git        │
│  - Files   │  │                            │  │  - Terminal   │
│  - Git     │  │     Dynamic Content        │  │  - Settings   │
```


## 关键 UI 组件

| 组件          | 路径                          | 说明              |
| ------------- | ----------------------------- | ----------------- |
| AppContent    | `src/components/app/`         | 应用主外壳        |
| MobileNav     | `src/components/app/`         | 移动端底部导航    |
| Header        | `src/components/app/`         | 顶部导航栏        |
| Sidebar       | `src/components/sidebar/`     | 侧边栏导航        |
| ChatView      | `src/components/chat/`        | 聊天主视图        |
| MessageList   | `src/components/chat/`        | 消息列表          |
| MessageInput  | `src/components/chat/`        | 消息输入框        |
| FileTree      | `src/components/file-tree/`   | 文件树组件        |
| CodeEditor    | `src/components/code-editor/` | CodeMirror 编辑器 |
| GitPanel      | `src/components/git-panel/`   | Git 面板          |
| Terminal      | `src/components/shell/`       | 终端组件          |

## 主题与样式

- 使用 Tailwind CSS 作为样式解决方案
- 支持明暗主题切换 (通过 Tailwind CSS 的 `dark` 类)
- 使用 CSS 变量管理主题色
- 响应式设计使用 Tailwind 的响应式前缀 (`md:`、`lg:` 等)
